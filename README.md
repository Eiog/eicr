# Eicr-多项目快速开始模板
by<https://github.com/Eiog>
# Install

```bash
npm install eicr -g
```
## Usage
```bash
eicr create <projectName>
```
## Options
```bash
eicr create <projectName>

eicr -v --version   🎈版本信息~
eicr -f --force     🕹️强制覆盖同名目录~
eicr -h --help      ❓帮助信息~
```
[模板列表](https://github.com/Eiog/eicr/blob/main/store.json)
## Features:

- Bundle with [tsup](https://github.com/egoist/tsup)
- Test with [vitest](https://vitest.dev)
