# Eicr-多项目快速开始模板
by<https://github.com/Eiog>
# Install

```
npm install eicr -g
```
## Arguments
```js
<name>  ✨ 项目名称~
```
## Options
```js
eicr <projectName> 

eicr -v --version   🎈版本信息~
eicr -f --force     🕹️强制覆盖同名目录~
eicr -h --help      ❓帮助信息~
```
